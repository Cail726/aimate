使用说明

API Key 怎么配？

调度器从系统环境变量或项目目录下的 .env 文件读取 API Key。

设置面板里只填环境变量名（如 DEEPSEEK_API_KEY），不填 Key 值本身。

Key 永远不会出现在页面上，不会泄露。

怎么设：setx DEEPSEEK_API_KEY "sk-xxx" 或在 D:\AIMate\.env 文件里写 DEEPSEEK_API_KEY=sk-xxx

怎么换模型？

点右上角 ⚙ 齿轮图标 打开设置面板，可以改：

  • 分析模型 — 负责拆解任务的 DeepSeek 模型，支持任意兼容 OpenAI API 的服务
  • Claude / Codex 命令 — 修改命令行参数，比如换模型、加参数

改完点保存即时生效，重启后依然保留。

费用怎么算的？

调度器只统计分析阶段的费用（调用 DeepSeek API 拆解任务），基于公开定价估算。

Claude 和 Codex 作为外部进程执行，费用无法统计，需各自查询账单。

点击 header 的 ≈ $x 徽章可查看每次分析的详细 token 和费用记录。

注意：费用为估算值，不包含税费/折扣，以 API 平台账单为准。

Header 上的指示灯

绿色实心 — 工具已安装，可用

红色空心闪烁 — 工具未找到，hover 看安装指引

Claude Code 和 Codex CLI 需要单独安装（npm 全局安装或下载安装包）。

工作流程

  1. 输入需求 → 点"分析需求"
  2. 查看拆解方案，确认哪些任务分给 Claude、哪些分给 Codex
  3. 点"执行方案"，实时看进度
  4. 查看结果，展开看详细输出
