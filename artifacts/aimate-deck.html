核心功能

▸ 智能拆解

自然语言输入，AI 分析后拆成 2-5 个独立子任务，标注每个任务适合哪个工具，给出理由

▸ 实时执行

SSE 推送进度，每个子任务的运行状态、耗时、输出预览一目了然

▸ Web 配置

点齿轮直接在浏览器改模型、temperature、API 端点。保存即时生效，不用碰配置文件

▸ 费用追踪

每次分析抓取 token 消耗，按定价估算费用。点击 ≈ $ 徽章查看历次明细

三步上手

零感调度器是一个 Python 本地 Web 服务，启动后在浏览器里操作

1

获取项目

GitHub 页面点 Code →
Download ZIP 下载
解压到任意文件夹
或 git clone 仓库地址

2

安装依赖 + 设 Key

双击 start.bat 自动安装
Python 包（fastapi 等）
在项目文件夹新建 .env
写入 DEEPSEEK_API_KEY=sk-你的key

3

启动

双击 start.bat 启动服务
自动打开浏览器
地址：127.0.0.1:7799
首次使用自动弹出帮助

需要提前装好 Claude Code 和 Codex CLI

可用零感安装器一键部署 Claude Code · Codex 支持即将加入
